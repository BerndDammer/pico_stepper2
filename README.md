# 
Trainee Project: Pico C-SDK; 2 Phase Stepper with Encoder for Closed loop
# HW
- Encoder Phase A GPIO2
- Encoder Phase B GPIO3
- L298N
- -INA+ GPIO16
- -INA- GPIO17
- -INB+ GPIO14
- -INB- GPIO15
# SW
Pico C SDK
- Console Application 
- StdIO over UART0 or USB
# Function
- Full Half WAve Stepping Timed Bitbang 
- Microstepping with PWM
